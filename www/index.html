funcgi

Welcome! Programs are served under /cgi-bin/, like this shell script.

Depending on your setup, you might have these examples available:

  • Forth
  • Guile
  • Jim
  • Lua
  • OCaml
  • Perl
  • Python
  • Ruby
  • Tcl

The purpose of this project is to show how simple and fun the web can and should be. Pull requests that add more fun (add more languages, flesh out examples, etc.) are welcome.
